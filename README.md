Run the project:
make run

Create Wallet:
    wallet create -n <walletName>

Delete Wallet:
    wallet delete -n <walletName>

Get Wallet:
    wallet get -n <walletName>

Wallet List:
    wallet list

Create Address:
    address create -w <walletName> -c <coinSymbol>

Address List:
    address list -w <walletName>
